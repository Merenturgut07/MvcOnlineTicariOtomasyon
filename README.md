# Projemizin adı AspNet MVc5 ile Online Ticari Otomasyon 
## Proje Amacı: 

-MVC 5 (.NET Çerçevesi)
</br>
-MSSQL Server
</br>
-Orm aracı olarak Entity Framework
</br>
-Code First yaklaşımı ile geliştirilmiştir.
</br>
-Session Kullanımı
</br>
-Güvenlikler için Authorize ve Authentication Kullandık
</br>
-Google Chart
</br>
-Sweet Alert
</br>
-Hata Sayfaları
</br>
-LINQ Sorguları
</br>
-QR Kod Oluşturma
</br>
